Poly Spider New Android Crawler

Poly Spider is mainly designed for Android app synchronizing and categorizing by focusing on grabing data from Android Markets.

View Demo on Github
WHAT WE DO

Data Crawling

Selected Android Markets have been crawled for Android App data collecting. Scrapy spiders have been used to crawl the market to achieve useful information by parsing the content with xpath.

Auto Categorizing

Intelligent categorizing method has benn applied for auto adjusting grabbed category information, which will be used for auto defining blacklist/whitelist policy of MAM based on category.

Data Analyzing

Charts&&Grids would be used for showing the processed data information statistically in the future. APIs would aslo be released out to the guys who are interested in the data we have.
